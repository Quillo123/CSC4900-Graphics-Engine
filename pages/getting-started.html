About

This is a graphics library created as part of an independent study course conducted through Seattle Pacific University and in partnership of Dr. Carlos Arias. This application was created for educational purposes only.

The Library is created in C++ using the OpenGL rendering pipeline. The API consists of the core set of materials needed for rendering 3D meshes, using a Unity-like scene management system.
